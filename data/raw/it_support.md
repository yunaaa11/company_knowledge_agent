# IT 故障处理指南

## 常见问题及解决方案

### 问题1：无法连接公司WiFi
**解决方案**：
1. 确认输入的密码正确（公司WiFi密码：Tech2024!）
2. 忘记网络后重新连接
3. 重启电脑和路由器
4. 联系IT支持分机：8888

### 问题2：VPN连接失败
**解决方案**：
1. 检查网络是否正常
2. 确认VPN账号密码正确（使用公司邮箱密码）
3. 切换VPN协议（UDP/TCP）
4. 重新安装VPN客户端

### 问题3：Outlook无法收发邮件
**解决方案**：
1. 检查网络连接
2. 确认邮箱容量未满（上限50GB）
3. 重启Outlook
4. 重建Outlook配置文件

## 故障报修流程

1. 拨打IT服务热线：400-888-XXXX
2. 或登录IT服务台：https://support.company.com
3. 填写工单（问题描述、截图、紧急程度）
4. IT人员2小时内响应
